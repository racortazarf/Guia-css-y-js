Funciones Javascript

Como todos los demás objetos, Function se puede crear utilizando la instrucción new: new Function ([arg1[, arg2[, ... argN]],] cuerpo-de-la-función)

arg1, arg2, ... argN Nombres para ser utilizados por la función como nombres formales de argumento. Cada uno de ellos debe ser una cadena que corresponde a un identificador válido JavaScript o una lista de esas cadenas separadas por una coma; por ejemplo "x", "elValor", o "a,b".
